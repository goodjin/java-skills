# Java Skills 项目

## 目标
通过学习10个优秀开源项目，沉淀 Java 开发最佳实践。

## 10个开源项目

| # | 项目 | GitHub | 类别 |
|---|------|--------|------|
| 1 | Spring Boot | spring-projects/spring-boot | Web框架 |
| 2 | Dubbo | apache/dubbo | RPC |
| 3 | RocketMQ | apache/rocketmq | 消息队列 |
| 4 | ShardingSphere | apache/shardingsphere | 数据库中间件 |
| 5 | SkyWalking | apache/skywalking | APM监控 |
| 6 | Redisson | redisson/redisson | Redis客户端 |
| 7 | Hutool | dromara/hutool | 工具库 |
| 8 | Guava | google/guava | 工具库 |
| 9 | MyBatis-Plus | baomidou/mybatis-plus | ORM |
| 10 | JeecgBoot | jeecgboot/jeecg-boot | 低代码平台 |

## 学习流程
1. 分析项目 → 生成 PRD
2. 编写实现 → 核心功能 Demo
3. 对比源码 → 学习最佳实践
4. 沉淀 Skills → 解决方案文档
