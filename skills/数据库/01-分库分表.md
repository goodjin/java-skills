# 分库分表

## 问题
如何实现数据分片？

## 分片算法

### 1. 哈希分片

```java
public int shard(String key, int dbCount, int tableCount) {
    return Math.abs(key.hashCode()) % (dbCount * tableCount);
}

public String getDB(int shardIndex) {
    return "db_" + (shardIndex % 4);
}

public String getTable(String tableName, int shardIndex) {
    return tableName + "_" + (shardIndex / 4);
}
```

### 2. 范围分片

```java
public String getDB(String id) {
    int num = Integer.parseInt(id);
    if (num <= 10000) return "db_0";
    if (num <= 50000) return "db_1";
    return "db_2";
}
```

### 3. 一致性哈希

```java
public class ConsistentHash {
    private TreeMap<Long, String> circle = new TreeMap<>();
    
    public void addNode(String node) {
        for (int i = 0; i < 160; i++) {
            long hash = hash("node-" + i);
            circle.put(hash, node);
        }
    }
    
    public String getNode(String key) {
        long hash = hash(key);
        return circle.ceilingEntry(hash).getValue();
    }
}
```

## 最佳实践

- 分片键选择: 避免跨库查询
- 分布式ID: Snowflake / Leaf
- 跨库聚合: 内存聚合 + 并行查询
- 扩容: 预先分片，避免迁移
