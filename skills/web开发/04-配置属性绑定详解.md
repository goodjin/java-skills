# Spring Boot @ConfigurationProperties 配置属性绑定详解

## 概述

`@ConfigurationProperties` 是 Spring Boot 提供的强大特性，用于将外部配置文件（如 application.properties 或 application.yml）中的属性绑定到 Java 对象中。它支持分层配置、类型安全的属性绑定、自动校验等功能。

## 源码分析

### 1. @ConfigurationProperties 注解定义

```java
// ConfigurationProperties.java
@Target({ElementType.TYPE, ElementType.METHOD})
@Retention(RetentionPolicy.RUNTIME)
@Documented
@Indexed
public @interface ConfigurationProperties {
    
    // 配置属性前缀
    @AliasFor("prefix")
    String value() default "";
    
    @AliasFor("value")
    String prefix() default "";
    
    // 忽略无效字段
    boolean ignoreInvalidFields() default false;
    
    // 忽略未知字段
    boolean ignoreUnknownFields() default true;
    
    // 验证配置
    boolean validate() default true;
    
    // 松散绑定
    boolean relaxNames() default true;
}
```

### 2. 配置属性绑定器 Binder

```java
// Binder.java - 核心绑定逻辑
public class Binder {
    
    public <T> T bind(ConfigurationPropertyName name, Bindable<T> target) {
        // 创建绑定结果
        BindResult<T> result = BindResult.of(getBindHandler(target));
        
        // 绑定属性
        // ...
        
        return result.orElse(null);
    }
    
    // 绑定过程
    private <T> void bindObject(ConfigurationPropertyName name, BeanPropertyBinding binding, 
            Bindable<T> target, Aggregator aggregator) {
        // 递归绑定对象的每个属性
        for (Map.Entry<String, Object> entry : properties.entrySet()) {
            ConfigurationPropertyName propertyName = ConfigurationPropertyName.of(entry.getKey());
            
            if (propertyName.startsWith(name)) {
                // 绑定子属性
                bindProperty(propertyName, target);
            }
        }
    }
}
```

## 核心类解释

### ConfigurationProperties

- **作用**: 标记类或方法进行配置属性绑定
- **属性**:
  - `prefix`: 配置属性前缀
  - `ignoreInvalidFields`: 忽略无效字段
  - `ignoreUnknownFields`: 忽略未知字段

### Binder

- **作用**: 将配置属性绑定到 Java 对象
- **核心方法**: `bind()`, `bindObject()`

### ConfigurationPropertyName

- **作用**: 表示配置属性名称
- **功能**: 支持松散绑定名称解析

### Bindable

- **作用**: 表示可绑定的目标
- **包含**: 类型信息、注解信息、泛型信息

## 代码示例

### 1. 基本用法

```java
// 配置属性类
@ConfigurationProperties(prefix = "myapp")
public class MyAppProperties {
    
    private String name;
    private int port;
    private boolean enabled;
    
    // getters/setters
}

// 启用配置属性
@SpringBootApplication
@EnableConfigurationProperties(MyAppProperties.class)
public class MyApplication {
    public static void main(String[] args) {
        SpringApplication.run(MyApplication.class, args);
    }
}

// 或直接注册为 Bean（Spring Boot 2.2+）
@Component
@ConfigurationProperties(prefix = "myapp")
public class MyAppProperties {
    // ...
}

// 使用配置属性
@Service
public class MyService {
    
    private final MyAppProperties properties;
    
    public MyService(MyAppProperties properties) {
        this.properties = properties;
        System.out.println("App name: " + properties.getName());
    }
}
```

### 4. 配置校验

```java
// 添加依赖
// <dependency>
//     <groupId>org.springframework.boot</groupId>
//     <artifactId>spring-boot-starter-validation</artifactId>
// </dependency>

@ConfigurationProperties(prefix = "myapp")
@Validated
public class ValidatedProperties {
    
    @NotBlank
    private String name;
    
    @Min(1)
    @Max(65535)
    private int port;
    
    @Pattern(regexp = "\\d{4}")
    private String code;
    
    @Valid
    private Server server = new Server();
    
    public static class Server {
        @NotBlank
        private String host;
        
        @NotNull
        private Integer port;
    }
}
```

### 5. 默认值

```java

public class DefaultProperties {
    
    // 基础类型有默认值

    private boolean enabled = true;
    private String name = "default";
    
    // 嵌套对象有默认值

    public static class Server {
        private String host;
        private int port;
        
        public Server() {}
        
        public Server(String host, int port) {
            this.host = host;
            this.port = port;
        }
    }
}
```

### 6. 构造函数绑定

```java
// 使用构造函数绑定（推荐）
@ConfigurationProperties(prefix = "myapp")
@ConstructorBinding
public class ConstructorProperties {
    
    private final String name;
    private final int port;
    private final Server server;
    
    public ConstructorProperties(
            @ConfigurationProperties(prefix = "myapp.server") Server server) {
        this.name = "default";
        this.port = 8080;
        this.server = server;
    }
    
    // 需要 @ConstructorBinding
}

// 或使用 record (Java 16+)
@ConfigurationProperties(prefix = "myapp")
@ConstructorBinding
public record MyAppProperties(String name, int port, Server server) {
    public Server {
        // 验证逻辑
    }
}
```

## 配置属性命名规则

| 配置格式 | 松散绑定 | 示例 |
|---------|---------|------|
| 短横线 | camelCase | `my-config-name` → `myConfigName` |
| 下划线 | camelCase | `MY_CONFIG_NAME` → `myConfigName` |
| 驼峰 | - | `myConfigName` |
| 全大写 | 下划线 | `MY_CONFIG_NAME` |

## 总结

`@ConfigurationProperties` 是 Spring Boot 最重要的特性之一，提供了：

1. **类型安全**: 编译时类型检查
2. **松散绑定**: 多种命名格式支持
3. **属性校验**: JSR-303/JSR-380 支持
4. **嵌套绑定**: 复杂配置结构支持
5. **默认值**: 合理设置默认值
6. **元数据**: IDE 智能提示支持
7. **刷新机制**: 动态刷新配置

合理使用配置属性可以大大提高配置管理的可维护性和代码质量。
