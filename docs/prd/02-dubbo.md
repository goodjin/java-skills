# Dubbo PRD

- **名称**: Dubbo
- **GitHub**: https://github.com/apache/dubbo
- **Star**: 40k+
- **公司**: 阿里巴巴

## 核心功能
1. 高性能 RPC 调用
2. 服务注册与发现
3. 负载均衡 (Random/RoundRobin/LeastActive)
4. 集群容错 (Failover/Failfast)
5. 熔断降级

- 掌握服务治理
