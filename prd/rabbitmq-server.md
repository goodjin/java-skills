# RabbitMQ Server 项目需求分析

## 项目定位（一句话）

功能丰富的多协议消息和流代理。

## 核心功能列表

- AMQP 1.0/0-9-1支持
- RabbitMQ Stream协议
- MQTT支持
- STOMP支持
- WebSocket支持
- 消息持久化
- 集群支持
- 分布式
- 插件系统

## 快速开始要点

- 下载发行版
- Erlang安装
- 启动服务

## 文档结构特点

- 官方文档详细
- 社区活跃
- 多种平台支持
