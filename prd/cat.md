# CAT 项目需求分析

## 项目定位（一句话）

美团点评开源的实时应用监控平台，提供全面的实时监控告警服务。

## 核心功能列表

- 实时处理（秒级数据）
- 全量数据采集
- 高可用监控
- 故障容忍
- 高吞吐能力
- 可扩展分布式部署
- 多语言客户端（Java、C/C++、Node.js、Python、Go）
- 监控模型：Transaction、Event、Heartbeat、Metric

## 快速开始要点

- 服务端项目
- 多语言客户端SDK
- 支持采样聚合
- 支持文件存储引擎
