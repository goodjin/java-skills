# Apache Kafka 项目需求分析

## 项目定位（一句话）

开源分布式事件流平台，用于高性能数据管道、流分析、数据集成等。

## 核心功能列表

- 分布式消息队列
- 高吞吐量
- 低延迟
- 持久化
- 水平扩展
- 消费者组
- 流处理
- 连接器框架
- Java 17+支持

## 快速开始要点

```bash
./gradlew jar
# 参考官方快速入门
```

- 需要Java 17+
- Scala 2.13
