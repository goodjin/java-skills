# Hystrix 项目需求分析

## 项目定位（一句话）

Netflix开源的分布式系统延迟和容错库（已处于维护模式）。

## 核心功能列表

- 延迟和容错
- 熔断器模式
- 降级处理
- 快速失败
- 线程隔离
- 信号量隔离
- 实时监控
- 配置更新

## 快速开始要点

- Maven引入
- Spring Cloud Netflix集成
- 注解驱动

## 文档结构特点

- Netflix开源
- 维护模式
- 文档详细
