# Pinpoint PRD 分析

## 项目概述

**项目名称**: Pinpoint  
**类型**: APM(应用性能管理)工具  
**开源协议**: Apache License 2.0

## 简介

Pinpoint是由韩国Naver团队开源的APM工具，为大规模分布式系统提供解决方案。受到Google Dapper的启发，Pinpoint通过追踪分布式应用中的事务，帮助分析系统整体结构和组件间关联。

## 核心特性

- **分布式追踪**: 跨服务调用链追踪
- **自动埋点**: 无侵入式字节码增强
- **应用拓扑**: 自动生成应用拓扑图
- **实时监控**: QPS、响应时间、错误率监控
- **多语言支持**: Java、PHP、Python Agent
- **可视化界面**: 直观的Web UI

## 适用场景

- 微服务性能分析
- 分布式调用链分析
- 系统瓶颈定位
- 性能问题排查

## 技术栈

- Agent: Java字节码插桩
- Collector: HBase存储
- Web UI: 前端可视化
