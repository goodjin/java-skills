# XXL-JOB PRD 分析

## 项目概述

**项目名称**: XXL-JOB  
**类型**: 分布式任务调度平台  
**开源协议**: GPLv3

## 简介

XXL-JOB是一个分布式任务调度平台，由个人开发者许雪里创建并维护。平台提供可视化的任务管理、调度监控、运行日志等功能，已被众多互联网公司采用。

## 核心特性

- **可视化控制台**: Web管理界面
- **任务管理**: 任务增删改查
- **调度中心**: 分布式任务调度
- **执行器**: 多种执行器支持
- **路由策略**: 故障转移、轮询等策略
- **任务分片**: 分布式任务分片执行
- **失败重试**: 失败自动重试
- **任务依赖**: 任务执行依赖支持

## 适用场景

- 定时任务调度
- 异步任务处理
- 数据同步任务
- 报表生成任务

## 技术栈

- Java后端
- Spring Boot
- MySQL存储
- 支持Docker部署
