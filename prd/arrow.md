# Apache Arrow 项目需求分析

## 项目定位（一句话）

通用的列式内存格式和多语言工具箱，用于快速数据交换和内存分析。

## 核心功能列表

- 列式数据格式（Columnar Format）
- IPC序列化格式（进程间通信）
- ADBC（Arrow Database Connectivity）
- Flight RPC协议
- 多语言库支持（C++、Java、Go、Python、R、Ruby、Swift、Rust等）
- 零拷贝内存管理
- 自描述二进制格式
- 与Apache Hadoop生态集成
- Gandiva（LLVM表达式编译器）

## 快速开始要点

- Maven依赖引入
- 支持Java 8+
- 与多种数据系统集成
