# Spring Security 项目需求分析

## 项目定位（一句话）

Spring生态的安全框架，提供认证、授权、加密等企业级安全服务。

## 核心功能列表

- 认证（Authentication）
- 授权（Authorization）
- 加密
- OAuth2支持
- SAML支持
- LDAP支持
- 方法级安全
- Web安全
- Spring Boot集成
- Java 17+要求

## 快速开始要点

- Spring Boot Starter引入
- Hello World示例快速入门
- 参考官方文档
