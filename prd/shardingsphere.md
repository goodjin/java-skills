# Apache ShardingSphere 项目需求分析

## 项目定位（一句话）

构建在异构数据库之上的分布式数据库生态系统，提供分片、读写分离等能力。

## 核心功能列表

- 数据分片
- 读写分离
- 分布式事务
- 分布式治理
- 数据库网关
- 多数据源访问
- SQL审计
- 性能增强

## 快速开始要点

- Maven引入
- Spring Boot集成
- 配置文件
