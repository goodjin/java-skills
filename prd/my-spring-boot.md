# My-Spring-Boot PRD 分析

## 项目概述

**项目名称**: My-Spring-Boot  
**类型**: Spring Boot自定义学习项目

## 简介

My-Spring-Boot是一个用于学习理解Spring Boot原理的项目。通过手动实现简化版Spring Boot，帮助理解自动配置、 starter机制等核心原理。

## 核心特性

- **自动配置**: 简化版自动配置实现
- **Starter机制**: 自定义starter示例
- **嵌入式容器**: 内嵌服务器支持
- **Spring Boot原理**: 核心机制学习

## 适用场景

- Spring Boot原理学习
- 框架设计借鉴
- 面试准备

## 技术栈

- Java基础
- Spring Framework
- 内嵌Tomcat/Jetty
