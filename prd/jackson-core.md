# Jackson Core 项目需求分析

## 项目定位（一句话）

Jackson数据处理器的核心底层流式解析和生成抽象库。

## 核心功能列表

- 流式JSON解析器
- 流式JSON生成器
- 支持多种数据格式（JSON、Smile、XML、CSV、YAML、Protobuf、CBOR）
- 高性能
- 低内存占用

## 快速开始要点

```xml
<dependency>
    <groupId>tools.jackson.core</groupId>
    <artifactId>jackson-core</artifactId>
    <version>3.x版本</version>
</dependency>
```
