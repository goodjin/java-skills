# Prometheus Client Java 项目需求分析

## 项目定位（一句话）

Prometheus监控系统的Java指标库。

## 核心功能列表

- 指标收集（Counter、Gauge、Histogram、Summary）
- 指标导出
- JVM指标
- Spring Boot集成
- 支持JDK 8+

## 快速开始要点

- Maven/Gradle引入依赖
- 定义指标
- 暴露HTTP端点

## 文档结构特点

- 官方文档：https://prometheus.github.io/client_java
- 活跃的社区
