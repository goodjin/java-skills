# Apache SkyWalking 项目需求分析

## 项目定位（一句话）

开源APM系统，为微服务、云原生和容器架构提供监控、追踪和诊断能力。

## 核心功能列表

- 分布式追踪
- 服务拓扑分析
- 服务中心可观测性
- 多语言探针（Java、.Net、PHP、NodeJS、Golang等）
- eBPF支持
- 高扩展性（支持千亿级遥测数据）
- 支持OpenTelemetry、Zipkin、Prometheus等
- 指标聚合
- 日志追踪

## 快速开始要点

- 探针引入
- OAP服务部署
- UI仪表盘

## 文档结构特点

- Apache项目
- APM领域成熟
- 中文社区活跃
