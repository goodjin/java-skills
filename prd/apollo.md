# Apollo 项目需求分析

## 项目定位（一句话）

可靠的分布式配置管理中心，支持多环境、多集群配置管理和热发布。

## 核心功能列表

- 统一配置管理（多环境、多集群、命名空间）
- 配置热发布（1秒内生效）
- 版本管理（支持回滚）
- 灰度发布
- 配置全文搜索
- 授权管理和发布审批
- 客户端监控
- 多语言SDK（Java、.Net、Go、Python、Node.js等）
- Open API平台
- 简单部署（只需Java+MySQL）

## 快速开始要点

- 基于Spring Boot和Spring Cloud开发
- Java SDK不依赖任何框架
- 支持Spring Placeholder、Annotation、ConfigurationProperties
- 一键部署脚本
