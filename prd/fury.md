# Apache Fory (Fury) 项目需求分析

## 项目定位（一句话）

高性能多语言序列化框架，基于JIT编译、零拷贝技术，性能提升高达170倍。

## 核心功能列表

- JIT编译序列化
- 静态代码生成
- 元数据打包和共享
- 跨语言序列化
- 行格式支持
- 零拷贝随机访问
- Arrow格式集成
- 引用保持
- 多态支持

## 快速开始要点

```xml
<dependency>
    <groupId>org.apache.fory</groupId>
    <artifactId>fory-core</artifactId>
    <version>最新版本</version>
</dependency>
```
