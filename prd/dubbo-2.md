# Dubbo-2 PRD 分析

## 项目概述

**项目名称**: Dubbo-2  
**类型**: RPC框架  
**开源协议**: Apache License 2.0

## 简介

Dubbo-2是Apache Dubbo的经典版本(2.x系列)，是高性能、轻量级的Java RPC框架。Dubbo提供了服务注册发现、负载均衡、容错、监控等微服务治理能力，广泛应用于国内互联网企业。

## 核心特性

- **服务注册与发现**: 支持Zookeeper、Nacos、Consul等注册中心
- **负载均衡**: 随机、轮询、一致性哈希等策略
- **容错机制**: 失败自动切换、失败安全、集群容错
- **RPC协议**: 支持Dubbo协议、HTTP、Hessian、gRPC等
- **扩展性**: 丰富的SPI扩展机制

## 适用场景

- 微服务架构下的服务间通信
- 分布式系统服务治理
- 高性能远程调用需求
- 企业级分布式应用

## 技术栈

- 纯Java实现
- Spring Boot/Starter支持
- Maven构建
