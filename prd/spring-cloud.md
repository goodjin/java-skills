# Spring Cloud 项目需求分析

## 项目定位（一句话）

Spring生态系统中的微服务构建工具，提供配置管理、服务发现、断路器等模式。

## 核心功能列表

- 配置管理（Spring Cloud Config）
- 服务发现（Eureka、Nacos）
- 断路器（Hystrix、Resilience4j）
- 负载均衡（Ribbon、LoadBalancer）
- API网关（Zuul、Gateway）
- 分布式追踪
- 消息驱动
- 安全

## 快速开始要点

- Spring Boot版本兼容
- 依赖版本管理
- 使用Spring Cloud Release

- Spring官方项目
- 多个子项目
