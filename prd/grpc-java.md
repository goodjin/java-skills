# gRPC Java 项目需求分析

## 项目定位（一句话）

高性能、开源的通用RPC框架的Java实现。

## 核心功能列表

- HTTP/2支持
- 多种语言互操作
- 流式处理
- 阻塞/非阻塞API
- 双向流
- TLS安全
- 支持Java 8+
- Android支持（minSdkVersion 21+）

## 快速开始要点

- Maven/Gradle依赖引入
- 定义proto文件
- 生成Java代码

## 文档结构特点

- 官方文档：https://grpc.io/
- 示例丰富
- 社区活跃
