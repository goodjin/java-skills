# Spring Cloud Gateway 项目需求分析

## 项目定位（一句话）

Spring Cloud的API网关，基于Spring WebFlux的响应式网关。

## 核心功能列表

- 动态路由
- 路由匹配（Path、Method、Header、Host等）
- 请求/响应过滤器
- 请求头/参数修改
- 路径重写
- 熔断器集成
- 服务发现集成
- Java 17、Spring Boot 3

## 快速开始要点

- Spring Boot Starter引入
- 配置文件或代码定义路由
- Predicate和Filter组合

## 文档结构特点

- Spring官方项目
- 文档详细
- 响应式架构
