# TestNG 项目需求分析

## 项目定位（一句话）

Java测试框架，灵感来自JUnit但具有更强大的功能。

## 核心功能列表

- 测试分组
- 依赖测试
- 参数化测试
- 注解驱动
- 多线程测试
- 数据驱动测试
- 监听器
- HTML报告
- Java 11+支持

## 快速开始要点

```xml
<dependency>
    <groupId>org.testng</groupId>
    <artifactId>testng</artifactId>
    <version>最新版本</version>
</dependency>
```
