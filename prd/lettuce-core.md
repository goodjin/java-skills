# Lettuce 项目需求分析

## 项目定位（一句话）

Redis的可扩展线程安全客户端，支持同步、异步和响应式使用。

## 核心功能列表

- 同步/异步/响应式API
- Redis Sentinel支持
- Redis Cluster支持
- 管道（Pipelining）
- 自动重连
- Redis数据结构支持
- Netty构建
- 线程安全

## 快速开始要点

```xml
<dependency>
    <groupId>io.lettuce</groupId>
    <artifactId>lettuce-core</artifactId>
    <version>最新版本</version>
</dependency>
```
