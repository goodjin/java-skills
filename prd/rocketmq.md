# Apache RocketMQ 项目需求分析

## 项目定位（一句话）

分布式消息和流媒体平台，具有低延迟、高性能和可靠性。

## 核心功能列表

- 发布/订阅消息模式
- 请求/Reply模式
- 流式处理
- 金融级事务消息
- 故障容忍和高可用
- 消息追踪
- 大数据和流生态集成
- 消息回溯
- 严格顺序消息
- 拉取和推送消费模型
- 多种消息协议（gRPC、MQTT、JMS、OpenMessaging）
- 分布式部署
- 批量消息
- SQL/Tag过滤
- 管理仪表盘

## 快速开始要点

- Maven引入客户端
- 支持Docker部署
