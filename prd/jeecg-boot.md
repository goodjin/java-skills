# JeecgBoot PRD 分析

## 项目概述

**项目名称**: JeecgBoot  
**类型**: 低代码开发平台 / AI低代码平台  
**开源协议**: Apache License 2.0

## 简介

JeecgBoot是企业级AI智能体与低代码平台，结合了AI生成能力与低代码开发模式。平台基于前后端分离架构，提供强大的代码生成器，支持一键生成前后端代码，大幅提升开发效率。

## 核心特性

- **低代码开发**: 在线表单、表单设计器
- **AI能力**: MCP协议支持、AI聊天生成代码
- **代码生成器**: 前后端一键代码生成
- **工作流引擎**: 在线流程设计
- **权限管理**: 按钮级、数据级权限控制
- **报表工具**: JimuReport零代码报表
- **微服务支持**: Spring Cloud Alibaba架构

## 适用场景

- 企业管理系统快速开发
- OA办公系统
- ERP、CRM系统构建
- 数据可视化大屏
- AI应用开发

## 技术栈

- 前端: Ant Design & Vue3
- 后端: Spring Boot 3 / Spring Cloud Alibaba
- 数据库: MyBatis-Plus
- AI: 支持ChatGPT、DeepSeek、Ollama
