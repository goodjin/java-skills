# Apache ZooKeeper 项目需求分析

## 项目定位（一句话）

分布式协调服务，用于配置管理、命名服务、分布式同步等。

## 核心功能列表

- 分布式协调
- 配置管理
- 命名服务
- 分布式锁
- Leader选举
- 集群管理
- 高可用

## 快速开始要点

- 下载发行版
- 配置zoo.cfg
- 启动服务

## 文档结构特点

- Apache项目
- 官方文档详细
- 广泛使用
