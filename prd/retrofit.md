# Retrofit 项目需求分析

## 项目定位（一句话）

Android和Java的类型安全HTTP客户端。

## 核心功能列表

- 类型安全HTTP客户端
- 注解定义请求
- 同步/异步支持
- Converter支持（JSON、XML等）
- OkHttp集成
- RxJava支持
- Spring Boot集成

## 快速开始要点

```xml
<dependency>
    <groupId>com.squareup.retrofit2</groupId>
    <artifactId>retrofit</artifactId>
    <version>最新版本</version>
</dependency>
```
