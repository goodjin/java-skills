# Zuul 项目需求分析

## 项目定位（一句话）

Netflix开源的L7应用网关，提供动态路由、监控、弹性、安全等功能。

## 核心功能列表

- 动态路由
- 请求过滤
- 负载均衡
- 静态响应处理
- 监控和追踪
- 安全控制
- Zuul 1/2双版本

## 快速开始要点

- Maven引入
- Spring Cloud Netflix集成
- 过滤器配置

## 文档结构特点

- Netflix开源
- Wiki文档详细
- 社区活跃
