# Swagger-Core PRD 分析

## 项目概述

**项目名称**: Swagger Core  
**类型**: OpenAPI规范实现  
**开源协议**: Apache License 2.0

## 简介

Swagger Core是OpenAPI规范的Java实现，提供了创建、描述API的工具。它是Swagger工具链的核心组件，支持OpenAPI 3.x规范。

## 核心特性

- **OpenAPI 3.1支持**: 最新规范支持
- **注解驱动**: 通过注解定义API
- **代码优先**: 从代码生成API文档
- **模型自动生成**: 自动扫描POJO生成模型
- **JAX-RS支持**: 完整的JAX-RS集成
- **Jakarta命名空间**: 支持javax和jakarta

## 适用场景

- REST API文档自动生成
- OpenAPI规范实现
- API设计与契约优先开发
- Swagger UI后端支持

## 版本要求

- Java 11+
- 支持JAX-RS 2.x

## 技术栈

- Java 11+
- JAX-RS实现
- Jackson
