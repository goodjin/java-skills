# Mockito 项目需求分析

## 项目定位（一句话）

Java最流行的Mock测试框架。

## 核心功能列表

- Mock对象创建
- Stub配置
- 验证调用
- 注解支持（@Mock、@Spy等）
- 静态方法Mock（Mockito 3+）
- 私有方法Mock
- 内联MockMaker
- 深度Stub
- Java 11+要求

## 快速开始要点

```xml
<dependency>
    <groupId>org.mockito</groupId>
    <artifactId>mockito-core</artifactId>
    <version>最新版本</version>
</dependency>
```
