# MyBatis 3 项目需求分析

## 项目定位（一句话）

Java SQL映射框架，简化对象与关系数据库的使用。

## 核心功能列表

- XML描述符或注解支持
- 存储过程支持
- SQL语句映射
- 对象关系映射
- 动态SQL
- 插件支持
- Spring集成
- Spring Boot Starter

## 快速开始要点

```xml
<dependency>
    <groupId>org.mybatis</groupId>
    <artifactId>mybatis</artifactId>
    <version>最新版本</version>
</dependency>
```
